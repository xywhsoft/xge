# XUI XSON Accordion Lab

`examples/xui_xson_accordion_lab` 验证 Accordion 的 XSON section、展开状态、模式和内置风格字段映射。

## 覆盖范围

- Accordion section 数量。
- 初始 expanded 状态。
- single 模式。
- header height 和 border color 等风格字段。

## 构建和运行

```bat
examples\xui_xson_accordion_lab\build.bat
build\xge_xui_xson_accordion_lab.exe --frames 2
```

## 通过标准

程序输出 `xui-xson-accordion-lab final-summary`，且 `load=1`、`sections=1`、`mode=1`、`style=1`。
