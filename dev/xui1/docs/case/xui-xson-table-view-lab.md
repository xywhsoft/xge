# XUI XSON TableView

`examples/xui_tableview_xson` 验证新 `tableView` XSON 字段：`columns`、`rows`、`cells`、`merges`、`selectionMode`、`scrollbarMode`、默认 metrics、基础颜色字段，以及 cell 对象中的 `tooltip`、`disabled`、`editing`、`invalid`、`dirty` 状态字段。

## 构建和运行

```bat
examples\xui_tableview_xson\build.bat
build\xui_tableview_xson.exe --frames 3
```

## 通过标准

程序输出 `xui_tableview_xson final-summary`，且 `create=1`、`layout=1`、`state=1`、`tableCount=4`。
