# 平台后端进展摘要

本文档汇总平台后端的实现状态、待验证项和阻塞项。详细实测记录仍只写入 `dev/docs/平台后端验证结果.md`。

## 状态分类

| 状态 | 含义 |
| --- | --- |
| 可本地开发验证 | 当前开发机可构建或运行基础流程。 |
| 脚手架存在 | 目录、API 或文档已规划，但需要目标环境继续实现。 |
| 待实机验证 | 需要真实设备、系统或浏览器环境验证。 |
| 低优先级 | 第一阶段不作为主要推进目标。 |

## 平台矩阵

| 平台 | 后端策略 | 图形基线 | 当前状态 | 主要阻塞 |
| --- | --- | --- | --- | --- |
| Windows | Sokol | OpenGL 3.3 | 可本地开发验证 | 需要持续回归 |
| Linux X11 | Sokol | OpenGL 3.3 | 待实机验证 | 发行版和驱动差异 |
| Linux Wayland | Sokol/后续 | OpenGL/GLES | 计划支持 | 后端接入和实机环境 |
| Linux EGL | EGL | GLES3 | 脚手架存在 | EGL display/surface 组合验证 |
| macOS | Sokol | OpenGL | 低优先级 | Apple OpenGL 限制 |
| Android Native | Sokol/NativeActivity | GLES3 | 脚手架存在 | 设备生命周期、assets、输入 |
| Android APK | Gradle/NDK | GLES3 | 计划脚手架 | APK 工程和资源接入 |
| iOS | Sokol | OpenGL ES | 低优先级 | SDK 限制和真机验证 |
| Web | Emscripten | WebGL2 | 脚手架存在 | 浏览器、资源、音频策略 |
| 小程序 | 自定义 bridge | WebGL2/GLES3 风格 | 核心目标 | 宿主桥接和平台 API |
| 板卡 Linux | EGL/GBM/KMS | GLES3 | 待实机验证 | 驱动、DRM 权限、输入 |
| 离屏 | EGL pbuffer/surfaceless | GLES3/OpenGL | 脚手架存在 | readback 和 render thread 验证 |
