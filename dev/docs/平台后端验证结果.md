# XGE 平台后端验证结果

本文档用于记录各平台实际构建和冒烟结果。只有对应平台结果标记为 `通过`，并保留环境、脚本、观察结果和关键日志后，才能关闭 `XGE_V2_SPEC.md` 中的平台运行路径任务。

可运行以下脚本汇总当前验证状态：

```text
check_platform_results.bat
./check_platform_results.sh
```

可用以下脚本追加验证记录：

```text
record_platform_result.bat "Windows" 通过 --env "Windows 11 / GPU / gcc" --scripts "build_test.bat" --summary "通过"
./record_platform_result.sh "Linux X11" 通过 --env "Ubuntu / X11 / Mesa" --scripts "./examples/platform_smoke/build_linux_x11.sh" --summary "通过"
```

可用以下脚本检查 platform smoke 日志是否包含 caps 与 runtime 快照：

```text
check_platform_smoke_log.bat smoke.log --require-final
./check_platform_smoke_log.sh smoke.log --require-final
```

状态：

- `未测`：尚未在目标平台执行。
- `通过`：完成构建、启动和 30 秒冒烟。
- `失败`：执行过，但存在构建、启动、渲染、输入或退出问题。
- `跳过`：当前机器缺少必要设备或工具链。
