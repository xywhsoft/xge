## 功能分组一

### `xgeFunctionName`

一句话说明这个函数。

**功能：**

说明这个函数做什么，适合在哪个场景使用，不适合解决什么问题。

**函数原型：**

```c
XGE_API int xgeFunctionName(type* pArg);
```

**参数：**

| 参数 | 方向 | 是否可为 `NULL` | 说明 |
| --- | --- | --- | --- |
| `pArg` | 输入 / 输出 | 否 | 参数含义、生命周期、所有权、有效范围、单位和默认值 |

**返回值：**

- `XGE_OK`：成功。
- 非 `XGE_OK`：失败，具体错误码见 `xge_result_t`，通常可通过日志或 xrt 最近错误获取更多信息。

**资源归属：**

- 说明该函数是否分配资源。
- 说明调用者应该用哪个 `Free` / `Unit` / `Close` / `Clear` 函数清理。
- 说明返回指针是借用还是调用者拥有。

**补充说明：**

- 调用顺序要求。
- 线程安全和重入注意事项。
- Sokol / EGL / WebGL / 小程序后端差异。
- 兼容性和版本注意事项。

**范例代码：**

```c
#include "xge.h"

int main(void)
{
	return 0;
}
```
