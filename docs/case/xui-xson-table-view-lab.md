# XUI XSON Table View Lab

`examples/xui_xson_table_view_lab` 验证 TableView 的 XSON columns、rows、adapter 和风格字段。

## 覆盖范围

- column 数量和对齐方式。
- row adapter 数据。
- selected 行。
- header height、row height、grid color 等风格字段。

## 构建和运行

```bat
examples\xui_xson_table_view_lab\build.bat
build\xge_xui_xson_table_view_lab.exe --frames 2
```

## 通过标准

程序输出 `xui-xson-table-view-lab final-summary`，且 `load=1`、`columns=1`、`rows=1`、`style=1`。
