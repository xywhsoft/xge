# XUI XSON Breadcrumb Lab

`examples/xui_xson_breadcrumb_lab` 验证 Breadcrumb 的 XSON 数据声明和选中状态映射。

## 覆盖范围

- XSON 页面加载。
- Breadcrumb segment 数量。
- segment id 字段。
- selected 索引映射。

## 构建和运行

```bat
examples\xui_xson_breadcrumb_lab\build.bat
build\xge_xui_xson_breadcrumb_lab.exe --frames 2
```

## 通过标准

程序输出 `xui-xson-breadcrumb-lab final-summary`，且 `load=1`、`segments=1`、`selected=1`。
