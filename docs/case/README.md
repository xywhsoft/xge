## 阅读建议

- 想先跑通窗口和渲染，从 `minimal-window.md` 开始。
- 想学习资源和纹理，读 `texture-sprite.md`。
- 想做工具、编辑器或 GUI，读 `xui-bridge.md`。
- 想做跨平台验证，读 `platform-smoke.md`。
- 想查函数细节时回 [API 索引](../api/README.md)。
