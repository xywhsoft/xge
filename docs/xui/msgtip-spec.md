# MsgTip Spec

## 目标

- [x] 提供单条轻量提示服务。
- [x] 支持无图标、信息、疑问、警告、错误类型。
- [x] 支持自定义图标纹理。
- [x] 支持自动过期和手动关闭。
- [x] 支持点击提示本体关闭。
- [x] 支持提示外区域事件穿透到下层控件。
- [x] 支持长文本按最大宽度换行。
- [x] 支持最小宽度、最大宽度、最小高度和垂直偏移配置。
- [x] 支持背景色、文字色和图标色配置。
- [x] 接入 Widget 的 event/update/paint 路径。
- [x] 不作为 XSON 控件节点提供。

## 非目标

- [ ] 不做通知队列；多条通知由 Toast 承担。
- [ ] 不做按钮和结果回调；需要决策时使用 MsgBox。
- [ ] 不做输入；需要输入时使用 InputBox。
- [ ] 不做 XSON `msgTip` 节点。

## 验证

- [x] `test/xui_msgtip_smoke.c` 覆盖初始化、宽度测量、换行、自动过期和点击关闭。
- [x] `examples/xui_msgtip` 覆盖人工观察和 frame-limit 自动摘要。
