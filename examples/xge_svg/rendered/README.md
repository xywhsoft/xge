# XGE SVG render references

This directory contains one current XGE render for every SVG case registered in
`tools/svg_compare/svg_compare_cases.ps1`.

The PNG files are generated by `examples/xge_svg/main.c` through
`tools/svg_compare/run_svg_compare.ps1`. They are learning and regression
references for XGE itself; they are not third-party comparison outputs.
